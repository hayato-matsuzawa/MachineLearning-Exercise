# 課題提出に関して．

- 課題は Moodle で提出する．
- 提出形式は原則として 各回でJupyter Notebook 形式の 1 ファイルとする．
- Notebook の冒頭に学籍番号・氏名を書くこと．
- Notebook の末尾に 100〜200 字程度の考察を書くこと．
- ファイル名は`dayN_学籍番号`であること（`N`は各回に対応）．スクリプトによる自動処理をするので，これに従わない場合に採点ミスの可能性がある．

---

# 提出内容

- 1 つの Notebook に，その回で指定した課題を順にまとめること．
- 各課題について，必要に応じて次を含めること．
- `コード`
- `実行結果`
- `図や表`
- `考察や解釈`
- 途中結果の提出が指定されている場合は，その数値や図も含めること．

---

# AI 利用について

- 生成 AI の相談利用は可とするが，出力の丸写し提出は不可．
- AI を使った場合は，Notebook の末尾に次の 2 点を 2〜4 行で書くこと．
- `何を質問したか`
- `どこを自分で修正・確認したか`
- 必要に応じて，提出内容について短く口頭確認することがある．

---

# 採点基準

- 4点満点（各 1 点）+ ボーナス（最大2点）を基本とする．
- `実行できる` 上から順に実行して最後まで動く
- `結果が妥当` 結果や数値が大きく外れていない
- `図と数値が揃っている` 必要な出力が確認できる
- `説明が自分の言葉で書かれている` 考察と必要なら AI 利用メモが書かれている
- `ボーナス要素がある` 必要に応じて加点．ボーナス要素がある場合はわかりやすく書くこと．
